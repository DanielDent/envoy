## Set up

`% npm install`

## Configure

Copy `run.sh.example` to `run.sh` and `chmod +x run.sh`. Then edit `run.sh` to use your account, API key, port and so on.

## Run

`ยง ./run.sh`

Note that the app expects the remote database to exist and be called `mbaas`.
